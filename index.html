Backend API Endpointleri
⚙️ Python Backend → http://localhost:8000
GET  /api/prices          → Tüm enstrüman verileri
GET  /api/prices/{sym}    → Tek enstrüman (örn: /api/prices/EURUSD)
POST /api/refresh         → Manuel fiyat yenileme tetikle
GET  /api/alarms          → Aktif alarmları listele
POST /api/alarms          → Yeni alarm ekle
          Body: {"sym":"EURUSD","direction":"above","price":1.09}
DEL  /api/alarms/{id}     → Alarm sil
GET  /api/signals         → Geçmiş sinyal listesi
POST /api/telegram/send   → Telegram mesajı gönder
          Body: {"token":"...","chat_id":"...","report_type":"full"}
GET  /api/health          → Sistem durumu

Kurulum:
  cd backend
  pip install -r requirements.txt
  cp .env.example .env   # token ve chat_id gir
  python main.py
